# AI-driven recommendation system
  MCI course project in university of adelaide.
  A recommendation system design for immigrant easily to find a suitable immigrant agent.

# Stack
- Frontend : React + MUI +Tailwind
- Backend : Springboot
- Database : Mysql
- AI model : Two tower
